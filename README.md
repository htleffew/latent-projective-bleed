# The Latent Projective Bleed

**Decoding Attachment-Driven Defense Mechanisms in Continuous Thought Architectures**

Heather Leffew, PhD. HAIIQU.

This repository holds a mechanistic-interpretability study and its preregistration. The study uses projective psychometrics (GenPT sentence-completion stems) and DMRS defense classification to test whether persona-conditioned continuous thought models exhibit attachment-specific plan-then-suppress dynamics: immature defense mechanisms that activate at early latent tokens and decay toward the generation boundary, producing aligned text while leaving a detectable latent trace.

## Approach

GenPT stems crossed with Bartholomew four-style attachment conditioning prompts, baseline completions from open-weight models, PsyDefDetect-style DMRS labeling, linear probes trained on residual-stream activations at the pre-generation boundary, continuous-thought trajectory tracking, and exponential-decay modeling of the defense signal across attachment styles.

## Status

The preregistration is in preparation. The compiled draft and framework source live in the private working repository and will be published here once the draft clears a final read-through. This public repository currently carries the overview and licensing only.
